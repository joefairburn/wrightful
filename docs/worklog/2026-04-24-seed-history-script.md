# 2026-04-24 тАФ `setup:local --history` for months of synthetic test-run data

## What changed

Added `--history` to `pnpm setup:local`. It synthesizes months of plausible test-run data and streams it through the existing ingest API so the history chart, flaky-tests page, branch filters, and run-list pagination have real shape to exercise тАФ no Playwright, just backdated runs generated locally. Implies `--no-fixtures` since it replaces the 3 canned Playwright scenarios with far more data.

To let the seeder backdate `createdAt` / `completedAt` across the history window, the open-run and complete-run handlers now accept optional override timestamps, but only when `import.meta.env.VITE_IS_DEV_SERVER` is true. In production both fields return 400.

- `--history-seed` (default `wrightful-seed-1` тАФ determinism for reruns)
